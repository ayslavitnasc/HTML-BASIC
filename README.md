# HTML BASIC

Um projeto simples de site estático desenvolvido em HTML puro, como parte do aprendizado inicial de desenvolvimento web.
Este projeto foi inspirado no roadmap.sh - Basic HTML Website Project.

📌 Objetivo
Praticar a estrutura básica de um documento HTML.

Entender como organizar conteúdo em páginas web.

Criar um site simples e funcional sem uso de CSS ou JavaScript.

🛠️ Tecnologias utilizadas
HTML5

🚀 Como visualizar
Clone este repositório:

bash
  git clone https://github.com/ayslavitnasc/HTML-BASIC.git
Abra o arquivo index.html em qualquer navegador.

📚 Aprendizados
Estrutura básica de um documento HTML (<!DOCTYPE html>, <html>, <head>, <body>).

Uso de tags semânticas como <header>, <main>, <footer>.

Criação de links entre páginas com <a>.
